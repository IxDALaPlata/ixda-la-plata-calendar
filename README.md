## Desarrollo

```bash
# checkout al branch `dev`
git checkout -b dev

# instalar dependencias con `npm`o `yarn`
# npm install
yarn install

# ambiente de desarrollo
npm run start-dev

# publicar a github pages (master)
npm run deploy
```
